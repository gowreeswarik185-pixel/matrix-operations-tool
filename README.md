# 🧮 Matrix Operations Tool (NumPy)

## 📌 Overview

The **Matrix Operations Tool** is a Python application built using the **NumPy** library. It performs common matrix operations such as addition, subtraction, multiplication, transpose, determinant calculation, inverse calculation, and other linear algebra operations.

This project demonstrates the practical use of NumPy for mathematical computations and serves as a beginner-friendly linear algebra project.

---

## ✨ Features

* Matrix Addition
* Matrix Subtraction
* Matrix Multiplication
* Matrix Transpose
* Determinant Calculation
* Matrix Inverse
* Matrix Shape Information
* User-Friendly Console Output
* Error Handling for Invalid Matrix Operations


### 2. Navigate to the project folder

```bash

```

### 3. Install NumPy

```bash
python -m pip install numpy
```

